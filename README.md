Global Finance Data Analysis – Excel Project

This project involves analyzing global finance datasets using advanced Excel techniques. It includes data cleaning, consolidation, PivotTables, PivotCharts, and dynamic dashboards to explore financial trends such as GDP growth, inflation rates, exchange rates, and market performance. The final Excel dashboard helps visualize patterns, compare regions, and highlight key financial indicators.
